# Creusot homepage

This is the source for Creusot's homepage https://creusot.rs

Command to build:

```
zola serve
```

## Subdomains

Subdomains of creusot.rs are hosted by other repositories:

- [devlog.creusot.rs](https://github.com/creusot-rs/devlog)
- doc.creusot.rs and guide.creusot.rs are both built from the main [Creusot repository](https://github.com/creusot-rs/creusot).
  The doc subdomain is deployed by a CI workflow of that repository.
  The guide subdomain is deployed via [yet another repository](https://github.com/creusot-rs/guide).
